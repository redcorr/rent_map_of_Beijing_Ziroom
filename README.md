### rent_map_of_Beijing_Ziroom
<a href='https://zhuanlan.zhihu.com/p/358985837/'>zhuanlan.zhihu.com/p/358985837</a><br>
数据包含2500元以下、合租、北京市的房租信息，时间为2021年3月。<br>
数据来源见：<a href='https://github.com/redcorr/crawler_ziroom'>github.com/redcorr/crawler_ziroom</a><br>
地图使用高德api，地图类型包含点图、热力图、蜂窝热力图。<br>
.py和.html文件需要配置不同的高德key，csv文件可能也需要编辑一下，加一列geo_code.<br>
